# ml-project

Starter repository for an ML pipeline project.

## Structure

- `src/components/`: ingestion, transformation, and training building blocks
- `src/pipeline/`: train and prediction pipeline entry points
- `src/exception.py`: shared exception helpers
- `src/logger.py`: shared logging setup
- `src/utils.py`: common utility functions

## Setup

```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install -e .
```

## Current Status

This repository is set up as a clean scaffold. The pipeline modules are placeholders and are ready for implementation.
